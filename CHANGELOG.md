# Changelog

## 3.0.0-alpha1

Initial alpha release of the MVEL3 Language Server and VS Code extension.

### Features
- Code completion for MVEL3 keywords
- Type completion for imported classes
- Method and field completion after `.` access
- Property access syntax sugar (e.g. `name` for `getName()`/`setName()`)
- Inline cast support (`expression#ClassName#`)
- Syntax highlighting for `.mvel` files
